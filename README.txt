PeriodIQ - 智能月經週期追蹤器

一個完全透明、使用者可控的月經週期追蹤應用程式

🌟 專案簡介
PeriodIQ 是一個創新的月經週期追蹤器，專注於資料隱私和使用者控制權。與傳統的追蹤應用不同，所有資料都儲存在使用者自己選擇的雲端平台上，確保完全的資料透明度和擁有權。
🎯 核心理念

資料透明: 所有資料都由使用者完全控制和可見
隱私至上: 資料儲存在使用者自己的雲端平台
跨平台同步: 支援多種雲端儲存服務
智能分析: 提供個人化的健康洞察和預測

🚀 快速開始
線上使用
直接開啟 index.html 檔案即可開始使用本地版本。
本地部署
bash# 克隆專案
git clone https://github.com/your-username/periodiq.git

# 進入專案目錄
cd periodiq

# 使用任何靜態伺服器
# 方法1: 使用 Python
python -m http.server 8000

# 方法2: 使用 Node.js
npx serve .

# 方法3: 使用 Live Server (VS Code 擴充功能)
📱 使用指南
首次設定

開啟應用程式
點擊「新增記錄」
輸入您的第一筆月經記錄
選擇相關症狀（可選）
添加備註（可選）

記錄月經週期

開始日期: 選擇月經開始的日期
結束日期: 選擇月經結束的日期（或讓系統自動預測）
症狀: 點擊相關症狀標籤
備註: 記錄任何特殊情況
點擊「記錄週期」儲存

查看分析

儀表板: 查看週期統計和當前狀態
月曆: 視覺化檢視歷史記錄和預測
洞察: 獲得個人化健康建議和趨勢分析

資料匯入
支援從其他月經追蹤應用匯入資料：

Clue 應用匯出的 JSON 檔案
Flo 應用匯出的 CSV 檔案
自定義格式的文字檔案
